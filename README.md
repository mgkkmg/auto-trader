## Commit Convention

- release: 배포 버전 업데이트 (major)
- feat: 신규 기능 추가 (minor)
- refactor: 코드 리팩토링 (minor)
- fix: 버그 수정 (patch)
- docs: 문서 수정 (patch)
- rename: 주석, 로그, 변수명 등 수정 (patch)
- style: 코드 포맷팅, 세미콜론 누락 (코드 변경 없는 경우) (patch)
- test: 테스트 코드, 리펙토링 테스트 코드 추가 (patch)
- chore: 빌드 업무 수정, 패키지 매니저 수정 (patch)
